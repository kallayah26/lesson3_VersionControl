Highlight #2 Short History of Git

For starters Git and Github are two separate things. Both however will be useful to us in this class. Git evolved from BitKeeper. Git was created in 2005 based on the foundation of the three characteristics that were just named. Highlight #3 Git Basics

Git serves as a version control database of sorts. It relies heavily on the use of local operations to do its job. All versions of a project are saved on the local disk when using Git. This allows users to make commit changes without wireless connection, which adds to the convenience of this tool. Git has amazing integrity features that protect files from corruption and getting lost. This helps preserve the time and work it takes to create your projects. I am interested in the basics of Git because it helps to understand the functions of a tool that I may be using to do some of the work for this course. It is relevant to LIS2360 because we are going to be using it to preserve our web apps during our time in this class.

Highlight #4 Getting a Git Repository

There are two ways to obtain a Git repository; turn a local directory into a Git repository or clone a Git repository that is already made. For this class we had to turn a local directory into a Git repository by connecting it to our Github account. The Set up process looks different for different PC devices, but ultimately they all serve the same purpose. The main git command is git init, this is used at the very beginning of the set up process and from there it is a matter of following a few other commands. I am interested in this because I just recently had to set up my first git repository and did it incorrectly, knowing this info is useful for future success. This course is going to rely heavily on the process of getting a git repository, we will have to do it for every assignment from here on out.

Creating a GitHub account is super easy and even more beneficial. GitHub serves as a host for the git repositories created over time for individual users. Once those are established GitHub is what you make of it. I am interested in GitHub because I believe that it will help me show case my capabilities at the completion of this course.
